A) Theme Install - top

Update Wordpress

To ensure a positive user experience, we highly recommend you update your wordpress to the latest stable version 3.4+. This will help to ensure that you are able to utilize all of the latest features of wordpress.

Installing GoodMinimal theme is just like installing any other wordpress theme. Its a simple process outlined in the 3 steps below.

  • Locate the good-minimal folder inside the zip file you downloaded from ThemeForest.
  • FTP Upload the good-minimal folder from our archive to your /wp-content/themes/ directory on your web server.
  • Activate the GoodMinimal theme by selecting the Appearance -> Theme tab and then activate it.

B) Blog Pages & Posts - top

The first step is to create site pages, and specify these pages in your Wordpress settings area.

  • Click on Pages > Add New and type the name Blog.
  • Select the template for this page over on the right in the Attributes section. Select Blog Template or Blog Full Width Template.

  • Enter the Page Headline in Additional page options, that will appear in your page box (optional).
  • Publish this page.
  • Click Posts in left sidebar.

  • Create Categories for blog posts.
  • Click Add New post. Enter title and content.
  • Enter Excerpt text.
  • Click Set Featured Image on the right in the Featured Image section.

  • Click Select Files button and select image to upload (upload the maximum size, image will be cropped and resized).

  • Click Use as featured image.

  • Publish this page.
  • Use widget Blog Sidebar and Blog Single Page Sidebar to show sidebars for Blog and Posts pages.

C) Contact Page - top

  • Click on Pages -> Add New and type the name Contact.
  • Enter content.
  • Select the template for this page over on the right in the Attributes section. Select Contact Page.

  • Enter the Page Headline (optional).

  • Publish this page.
  • To set up email where you want to receive emails please go to GoodMinimal Options -> General Theme Settings and add your email.

D) Portfolio Pages and Posts - top

  • Click on Pages -> Add New and type the name Portfolio.
  • Enter content.
  • Select the template for this page over on the right in the Attributes section. Select one of 5 portfolio templates Portfolio Categorized, Portfolio Switchable, Portfolio Two Columns, Portfolio Three Columns, Portfolio Four Columns.
  • Enter the Page Headline (optional).
  • Publish this page.
  • Click Portfolio in left sidebar.

  • Click Add New portfolio. Enter title and content.
  • Enter Excerpt text.
  • Click Set Featured Image on the right in the Featured Image section.

  • Click Select Files button and select the image to upload (upload the maximum size, image will be cropped and resized).

  • Click Use as featured image.

  • To show video (vimeo, youtube) in Portfolio on image click then enter Video URl in Portfolio Options like in this screenshot:

    Examples of url to include in this input cell:
    Another image -> http://www.sitename.com/images_fullscreen/2.jpg
    Youtube content -> http://www.youtube.com/watch?v=qqXi8WmQ_WM
    Vimeo content -> http://vimeo.com/8245346
    QuickTime movies -> http://trailers.apple.com/movies/universal/despicableme/despicableme-tlr1_r640s.mov?width=640&height=360
    External site -> http://www.google.com?iframe=true&width=100%&height=100%

  • To show gallery on image click in lightboox then upload images into Portfolio item with Media Gallery.

  • Publish this page.
  • Use unlimited portfolio pages: after you will create more portfolio items and will assign categories for them, you can create more Portfolio pages and to show there only categories that you want. After go to GoodMinimal Options -> Portfolio Settings and assign every Portfolio Page to a category. To show all portfolio items from in portfolio page just leave "Select category and save changes.

Note: On the homepage we list last 4 Portfolio items. In GoodMinimal Options -> Portfolio Settings you can hide portfolio section by check the checkbox Hide Portfolio from Homepage.

E) Homepage Slider - top

  • Click Slider in left sidebar.

  • Click Add New slider.
  • Click Set Featured Image on the right in the Featured Image section.

  • Click Select Files button and select image to upload (minimum width and height 960x447px for Flex Slider). If uploaded images will be greater, then we will crop them to 960x447px.

  • Click Use as featured image.

  • Add slide URL and Description (optional).

  • Publish this page.

F) Widgets Sidebars - top

GoodMinimal theme have unlimited sidebars for widgets. Default sidebars are 11. All these sidebars you can find in Appearance -> Widgets.

  1. Homepage Widgets Area
  2. Blog Sidebar
  3. Blog Single Page Sidebar
  4. Page Sidebar
  5. Portfolio Sidebar
  6. Contact Sidebar
  7. Contact Form
  8. Footer Column 1
  9. Footer Column 2
  10. Footer Column 3
  11. Footer Column 4

To create and use unlimited sidebars is very simple. In GoodMinimal Options you must to create sidebars name and in select them in page templates, portfolio items, blog posts. Please see 2 images below where you can create additional sidebars and use them.

Create sidebars in GoodMinimal Options -> Sidebars. Click button Add New Sidebar, enter sidebar name and click button Save changes. To delete one sidebar, just leave empty the input, delete sidebar name and click button Save changes.

When you editing the page please find option Select Sidebar from section Additional page options.
